## 🌟 Overview

EduMentor AI is a modern, responsive web application that revolutionizes education through artificial intelligence. Our platform provides personalized learning experiences, intelligent tutoring, and comprehensive educational support to help students worldwide achieve their academic goals.

### 🎯 Mission
To democratize quality education by making AI-powered personalized learning accessible to everyone, everywhere, supporting the United Nations Sustainable Development Goal 4 (Quality Education).

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for AI functionality
- Text editor (VS Code recommended)

### Installation

1. **Clone or Download**
   ```bash
   # If using Git
   git clone https://github.com/yourusername/edumentor-ai.git
   cd edumentor-ai

   # Or simply download the ZIP file and extract
   ```

2. **Open in Browser**
   - Open `index.html` directly in your browser, OR
   - Use a local server for better functionality

3. **Run Local Server (Recommended)**
   ```bash
   # Using Python (built-in)
   python -m http.server 8000

   # Using Node.js (if available)
   npx serve .

   # Using PHP (if available)
   php -S localhost:8000
   ```

4. **Access the Website**
   - Open `http://localhost:8000` in your browser
   - The website will load with all features functional

## 🔧 Customization

### Theme Customization
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #06b6d4;
    /* ... other variables */
}
```

### AI Model Configuration
Modify the AI integration in `script.js`:
```javascript
// Change AI model or parameters
const response = await fetch('https://api-inference.huggingface.co/models/YOUR_MODEL', {
    // ... configuration
});
```
